# Reelm — release feed

(Formerly OpenStream — this repo kept its original name for continuity, but
the app is now called **Reelm**.) This is the auto-update feed for Reelm, a
Stremio-protocol-compatible media client for Windows and Android TV. It holds
compiled installers/APKs + version manifests only — no source code.

## Download

- **Windows**: grab `Reelm_<version>_x64-setup.exe` from the
  [Latest release](https://github.com/Albrim98/openstream-releases/releases/latest).
- **Android / Google TV / Fire TV**:
  [Reelm_0.2.78_androidtv.apk](https://github.com/Albrim98/openstream-releases/releases/download/v0.2.78/Reelm_0.2.78_androidtv.apk) —
  sideload with the [Downloader](https://www.aftvnews.com/downloader/) app by
  entering that URL directly. The filename still says "0.2.78" but the file
  itself is kept up to date on every Android build — it's a fixed, permanent
  download link, not an old version.

After first install, both apps check for updates automatically.
